== Description ==

Configure WordPress to send emails via MailWish’s SMTP service. Get blazing-fast, inbox-focused email delivery at just $0.10 per 1,000 emails.

== Installation ==

1. Upload the plugin to `/wp-content/plugins/`.
2. Activate the plugin via the Plugins menu.
3. Go to **Settings → MailWish SMTP** and enter your SMTP credentials.
