== Installation ==

1. Upload the plugin files to `/wp-content/plugins/mailwish-smtp-wordpress/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings > MailWish SMTP to configure the plugin
4. Get your SMTP credentials from [https://mailwish.com](https://mailwish.com)
5. Enter your credentials and test the configuration
